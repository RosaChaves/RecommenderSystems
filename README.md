# RecommenderSystems
Deep Learning; pyspark for recommender systems
These codes summarize the udemy course https://www.udemy.com/course/recommender-systems/ 
1. Input data is rating.csv from the movielens-20m-dataset (Kaggle)
2. Data is preprocessed, shrinked etc
3. Collaborative Filtering, RBM and Matrix Factorization are explained
4. Codes for pyspark run from the terminal or from Pycharm
5. How to use keras
